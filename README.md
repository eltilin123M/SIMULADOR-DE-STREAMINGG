## ¿Qué es?
Es un programa de C# que evalúa contenidos para decidir si puede publicarse en una plataforma de streaming.

## ¿Cómo ejecutar el programa?
1. Abrir Visual Studio
2. Abrir el archivo Program.cs
3. Presionar F5

## ¿Que hace el programa?
El programa pide informacion sobre un contenido con el tipo, duración, clasificación, hora y nivel de producción
Con esos datos se decide si se puede publicar o no.

## Opciones
1. Evalular contenido
2. Ver reglas
3. Ver estadisticas
4. Reiniciar estadisticas
5. Salir
